# SQL скрипти для ЛР №4

## init_insert.sql
Для більш доцільного використання 
всіх інших команд доповнив таблиці новими полями .

## aggregation1.sql
Рахує кількість книг кожного жанру з наявних.

## aggregation2.sql
Рахує кількість книг, написаних кожним автором.

## aggregation3.sql
Рахує кількість наявних копій кожної книги.

## aggregation4.sql
Показує, скільки екземплярів книг
числиться у бібліотеці в цілому.

## join1.sql
Використовує INNER JOIN, показує книги 
разом з їхніми авторами.

## join2.sql
Використовує LEFT JOIN, показує книги 
разом з їхніми авторами, але також включає
авторів, чиїх книг немає в наявності.

## join3.sql
Використовує CROSS JOIN, показує імена 
клієнтів з іменами авторів.

## subquery1.sql
Показує книги, які зберігаються у бібліотеці 
більше, ніж у двох екземплярах.

## subquery2.sql
Показує здані копії книг.

## subquery3.sql
Показує доступні екземпляри книг.
